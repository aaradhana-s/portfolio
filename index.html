Food Waste Management System

Designed the complete database schema from scratch, including entity-relationship diagrams, normalized tables, and primary/foreign key relationships to keep the data consistent and reliable. Blind Stick

Built a fully working assistive device to help visually impaired individuals navigate safely by detecting nearby obstacles. The stick is fitted with an ultrasonic sensor that continuously scans the path ahead, calculating distance based on how long it takes sound waves to bounce back off an object. When an obstacle is detected within close range, an Arduino-controlled buzzer triggers an audio alert, giving the user real-time feedback to adjust their path before making contact. Worked as part of a team to design, wire, and physically mount the electronics onto a real walking stick to create a usable device.

Book My Stay App

Built a Java-based hotel booking system as part of a team project, applying core data structures to solve real booking challenges. Implemented a fair, FIFO-based request queue to handle multiple simultaneous bookings, a room allocation service that assigns unique room IDs while preventing double-booking, and a cancellation system with rollback history to safely restore inventory. Also added input validation with custom exceptions, file-based data persistence for recovering inventory state, and multithreaded booking processing using synchronization to ensure thread-safe access to shared inventory.
